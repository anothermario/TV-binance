Settings

Bot Configuration

Bot settings are managed via environment variables. Set the following variables to configure the bot:

  • BINANCE_API_KEY — Your Binance API key
  • BINANCE_API_SECRET — Your Binance API secret
  • WEBHOOK_PASSPHRASE — Passphrase used to authenticate TradingView webhook alerts
  • DB_PATH — Path to the SQLite database file (default: trades.db)
